# cocktail-vertical-slider

```
Al iniciar la pagina, se debera cargar un coctel aleatorio y guardarlo en una lista.

Si solo se tiene un elemento dentro de la lista de cocteles cargados, el boton hacia abajo debera estar desactivado.

Al dar click sobre el boton hacia arriba, debera cargarse otro coctel aleatorio y guardarlo en la lista de cocteles.

Si hay 2 o mas cocteles dentro de la lista, el boton hacia abajo debera mostrar el coctel que se encuentre en la
posicion anterior.

Si hay 2 o mas cocteles dentro de la lista y el coctel que se muestra es la primera posicion, el boton hacia abajo
debera cargar el ultimo coctel guardado en la lista.

Si se encuentra en una posicion que no sea la ultima guardada en la lista, el boton hacia arriba debera mostrar el
coctel que sigue en la lista y no uno aleatorio.

Si se encuentra en la ultima posicion y no hay un maximo de 5 cocteles guardados, el boton hacia arriba debera
cargar un nuevo coctel aleatorio.

Si se encuentra en la ultima posicion y hay un maximo de 5 cocteles guardados, el boton hacia arriba debera
cargar el coctel que se encuentra guardado en la primera posicion.
```
